Installation Instructions

Installation is done using GNU Autoconf. You simply have to fetch the package from Subversion and use the standard commands as shown below. Adjust the "--prefix" to your preferred installation path.

svn co http://mono-soc-2007.googlecode.com/svn/trunk/brian/FastCgi/ fastcgi-mono-server
cd fastcgi-mono-server
./autogen.sh --prefix=/usr && make && sudo make install

Server Specific Configuration

NOTE

The following documentation is currently based strictly on redirecting known ASP.NET extensions to the FastCGI Mono Server. This yields a performance advantage by not sending static and non-ASP.NET files across FastCGI, but provides a security disadvantage as it doesn't incorporate all of ASP.NET's file rejection. For example, ASP.NET would reject certain paths, like "App_Data/", by default.

It is always important to test that files containing secure data are not accessible in a web browser. For extra security, look in your web server's documentation for how to send all requests for a path straight to a FastCGI interpreter.
